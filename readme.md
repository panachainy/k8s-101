# K8S-101

## Pre-requisite

* Homebrew
* macOS
* Basic docker knowledge

## Setup

1. ทำการอัพเดท Homebrew

   ```bash
    brew update
   ```

2. ติดตั้ง minikube

   ```bash
    brew install minikube
   ```

   > kubectl จะติดตั้งไปพร้อมกับ minikube ไปด้วยเลยครับ

3. start minikube

   ```bash
    minikube start
   ```
